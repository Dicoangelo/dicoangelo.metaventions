# Client Project: The Signature Event

## Overview

Designed and built a luxury event marketing website for Cooper Swanson Ventures (Jordan Cooper & Leif Swanson) — an invitation-only networking gathering of 40 curated professionals at a private Miami penthouse (500 Brickell). Event took place February 20, 2026.

## Client

Cooper Swanson Ventures — Jordan Cooper and Leif Swanson, dental practice owners and luxury event hosts based in Miami, FL.

## Live URL

https://thesignatureevent.metaventionsai.com

## Deliverables

| Deliverable | Details |
|-------------|---------|
| **Marketing Website** | Full Next.js 16 site with hero, video showcase, 57-photo gallery, sponsor tiers, RSVP integration |
| **Post-Event Recap** | Transition from pre-event (countdown, invitation request) to post-event (video recap, photo gallery, next event CTA) |
| **Print Collateral** | Python-generated flyers, QR code generators for event registration |
| **Sponsor Integration** | Tiered sponsor system: title (MB2 Dental), premium (Bank OZK, Stefania Mogollon), beverage (Peroni), with ticker and grid display |
| **Analytics** | Vercel Analytics with custom CTA tracking (hero, nav, footer click events) |

## Technical Architecture

- **Framework:** Next.js 16, React 19, TypeScript 5
- **Styling:** Tailwind CSS 4, Framer Motion 12 animations
- **Design System:** Art deco luxury aesthetic — gold shimmer, grain textures, Playfair Display + Inter fonts
- **Video:** Vercel Blob Storage CDN, auto-play on scroll with volume controls
- **Gallery:** 57-photo lightbox with keyboard navigation, dual auto-scrolling rows
- **Deployment:** Vercel with auto-deploy from GitHub

## Key Outcomes

- Delivered end-to-end in under 2 weeks from initial client meeting to live deployment
- Zero backend complexity — pure static marketing site, no database, no API keys required
- Event successfully hosted 40+ curated professionals at 500 Brickell penthouse
- Post-event site continues to serve as portfolio piece and next-event funnel
- 15 React components, fully responsive, art deco design language
