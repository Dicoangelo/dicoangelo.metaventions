CRITICAL ISSUE: CSP BLOCKING SCRIPTS
================================================================================

PROBLEM:
  Content Security Policy uses 'strict-dynamic' which requires all scripts
  to have a special token (nonce). Next.js scripts don't have this token,
  so they're all being blocked.

IMPACT:
  • Site loads ✅
  • Content displays ✅
  • JavaScript blocked ❌
  • Interactive features disabled ❌
  • Chat doesn't work ❌
  • JD Analyzer doesn't work ❌
  • Forms not interactive ❌

SOLUTION:
  Remove 'strict-dynamic' from /src/middleware.ts (1-30 minutes)
  See: CSP_FIX_GUIDE.md for step-by-step instructions

SEVERITY:         CRITICAL
EFFORT:           VERY EASY (1-2 lines of code)
